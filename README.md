# AI 궁디팡팡 🖐️

Gemini, GPT, Claude — 세 AI 캐릭터의 엉덩이를 눌러 스트레스를 해소하는 인터랙티브 웹 게임입니다.

## 사용법

```bash
# 별도 설치 없이 브라우저에서 바로 실행
open index.html
```

또는 `index.html` 파일을 브라우저에 드래그 & 드롭하세요.

## 기능

- 캐릭터 하단 핫존 클릭 시 팡팡 애니메이션 + 말풍선 반응
- AI별 개별 카운터 및 합계 표시
- 레벨 시스템 (0→5→15→30회 이상 시 카드 상태 변화)
- 콤보 시스템 (5 / 10 / 20 연속 콤보 달성 시 이펙트)
- 팡팡 랭킹 보기 (🥇🥈🥉)
- 결과 화면 PNG 저장
